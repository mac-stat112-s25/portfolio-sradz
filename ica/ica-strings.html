# Optional: When we want to split a column into 2+ new columns (or do other types of string processing), but thereâ€™s no consistent pattern by which to do this, we can use regular expressions 

# (?<=[SP|FA]): any character *before* the split point is a "SP" or "FA"
# (?=2): the first character *after* the split point is a 2
